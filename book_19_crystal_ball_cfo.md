## Resource 2: Forecasting Accuracy Metrics

### Key Metrics

**Mean Absolute Error (MAE):**
```
MAE = (1/n) × Σ|Actual - Forecast|
```
- Average absolute difference
- Easy to interpret
- Same units as data

**Mean Absolute Percentage Error (MAPE):**
```
MAPE = (1/n) × Σ|(Actual - Forecast) / Actual| × 100
```
- Percentage error
- Comparable across scales
- Watch for division by zero

**Root Mean Square Error (RMSE):**
```
RMSE = √[(1/n) × Σ(Actual - Forecast)²]
```
- Penalizes large errors
- Good for outliers matter
- Same units as data

**Mean Percentage Error (MPE):**
```
MPE = (1/n) × Σ[(Actual - Forecast) / Actual] × 100
```
- Shows bias direction
- Positive = under-forecast
- Negative = over-forecast

### Benchmark Targets

| Forecast Type | MAPE Target | Notes |
|--------------|-------------|-------|
| Short-term cash flow | 5-10% | Daily/weekly |
| Monthly revenue | 10-15% | Established business |
| Quarterly revenue | 15-20% | Including seasonality |
| Annual revenue | 10-15% | Longer horizon |
| Expenses | 5-10% | More predictable |

### Tracking and Improvement

**Regular Review:**
- Monthly accuracy assessment
- Bias identification
- Pattern analysis
- Model adjustment

**Root Cause Analysis:**
- Large error investigation
- Assumption validation

- Customer concentration > 30%

- Days cash on hand < 15
